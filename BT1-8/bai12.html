  • Học lập trình Web

  • PHP
      PHP (viết tắt của cụm từ Personal Home Page) là ngôn ngữ lập trình
      kịch bản (scripting language) mã nguồn mở được dùng phổ biến để ra
      tạo các ứng dụng web chạy trên máy chủ
  • Javascript
      Javascript là một ngôn ngữ thông dịch, chương trình nguồn của nó
      được nhúng hoặc tích hợp vào tập tin HTML.
